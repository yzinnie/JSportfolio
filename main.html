SAN-CHECK (산-쳌)

산책 만족도를 위해 항상 체크하는 것들을 한 번에 체크하는 공간
REST API의 경험으로 작동 방식에 대해 공부할 수 있었던 프로젝트입니다.

1인 프로젝트

  • openweathermap API - 날씨 정보를 불러오는 API와 현재 위치 정보에 접근할 수 있는 API(Geolocation API)를 사용하여 사용자 위치에 맞는 현재 날씨를 출력하여 사용했습니다.
  • kakao API - REST API를 사용하여 현재 위치의 좌표와 블로그 글 자동 검색 기능을 구현했으며 MAP API를 사용하여 현재 위치와 지도 이미지를 구현했습니다.
  • 한국환경공단 API (미세먼지) - 카카오에서 제공하는 REST API를 사용하여 현재 위치의 좌표를 변환하고 행정구역 정보를 받아와 미세먼지 정보를 사용했습니다.
  • localStorage - 체크리스트 작성 및 삭제 기능을 구현했습니다.
  • firebase - 배포 및 배포 테스트를 했습니다.
  • 개발 서버에서 만난 CORS 에러에 proxy 서버를 사용하여 에러를 해결한 경험으로 CORS에 대응하는 방법에 대해 익힐 수 있었습니다.
  • 배포된 https에 암호화되지 않은 http 사이트에서 요청을 보낼 경우 발생하는 Mixed Content 에러로 보안의 중요성을 이해했습니다.

    • 배포링크 깃허브 에러노트
