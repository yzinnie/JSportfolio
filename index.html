* 사용한 API

  • Open Weather API
    날씨 정보를 불러오는 API와 현재 위치 정보에 접근할 수 있는 Geolocation API를 사용하여 사용자 위치에 맞는 현재 날씨를 출력
    API에서 제공하는 날씨 아이콘을 사용했습니다.
  • kakao API
    카카오의 REST API와 MAP API를 사용했습니다.
    지도를 화면상에 구현하기 위해 Geolocation API를 MAP API에 적용했으며 카카오에서 제공하는 깃발 아이콘을 사용했습니다.
    블로그 검색하기 기능으로 쿼리 파라미터에 '반려동물' 키워드를 넣어 자동으로 검색된 블로그 글을 제공합니다.
    현재 위치의 위경도를 TM 좌표로 변환했습니다.
  • 한국환경공단 API (미세먼지)
    TM 좌표를 입력하여 입력된 자표 주변 측정소 정보를 조회할 수 있는 기능을 사용했습니다.
    TM 좌표 사용을 위해 카카오 REST API를 사용하여 현재 위치의 위경도를 TM 좌표로 변환하고
    한국환경공단에서 제공하는 행정구역 정보를 받아와 미세먼지 정보를 구현했습니다.

  • firebase

  • 개발 서버에서 만난 CORS 에러에 proxy 서버를 사용하여 에러를 해결한 경험으로 교차 출처 리소스 공유에 대해 공부하는 경험이 되었습니다.
